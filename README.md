## Тестовое задание для RocketSales | Backend
Стек: TypeScript, NestJS

## Как установить?

1. Клонировать проект
2. Установить зависимости
```bash
$ npm install
```
3. Конфигурировать подключение к amoCRM в файле ```.env```

## Как запустить?

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
